Epoint Ödəniş Sisteminin İşləmə Prinsipi

Giriş

Epoint sistemi ödəniş qəbulunu saytınıza və tətbiqinizə bağlamağa imkan verir. Ödəniş düyməsini saytınıza əlavə etmək üçün Epoint.az saytındaki şəxsi kabinetinizdə öz saytınızın redirect URL-lərini əlavə etməlisiniz.

Tacir Yaratmaq üçün Tələb Olunan Məlumatlar

Epoint.az saytında tacir yaratmaq üçün bizə aşağıdakı məlumatları təqdim etməlisiniz:

  • Saytınızın ünvanı
  • Uğurlu ödəniş səhifəsinin URL-i (success_url)
  • Uğursuz ödəniş səhifəsinin URL-i (error_url)
  • Ödəniş nəticəsini göndərmək üçün URL (result_url)

Bu məlumatlar yoxladıqdan sonra sizə giriş açarları veriləcək: public_key (sistemdəki tacir identifikatoru) və private_key (API giriş açarı).

Ödəniş Axını

  1. Sorğunun yerinə yetirilməsi nəticəsində müştəri bankın ödəniş səhifəsinə yönəldiləcək.
  2. Müştəri kart rekvizitlərini doldurur və ödənişi təsdiqləyir.
  3. Uğurlu ödənişdə müştəri success_url-ə, əks halda error_url-ə yönləndirilir.
  4. Ödənişin təfərrüatları sizin qeyd etdiyiniz result_url-ə göndəriləcək.

API Sorğusunun Formalaşdırılması

Sorğunu formalaşdırmaq üçün data və signature parametrlərini POST metodu ilə aşağıdakı ünvanlara göndərin:

POST https://epoint.az/api/1/request

Müştərini isə bu ünvana yönəldin:

GET https://epoint.az/api/1/checkout

Parametrlərin İzahı

  • data — base64_encode(json_string) funksiyası ilə kodlaşdırılmış API parametrlərini özündə birləşdirən json sətridir.
  • signature — base64_encode(sha1(private_key + data + private_key, 1)) — hər sorğunun unikal yazısıdır.
  • base64_encode — MIME base64 formatında kodlaşdırılmış sətir qaytarır.
  • sha1 — 20 simvoldan ibarət hash sətri qaytarır.

Callback Funksiyasının İcra Edilməsi

Əməliyyat Epoint tərəfindən işləndikdən sonra data və signature parametrləri ilə POST sorğusu serverinizə (result_url) göndəriləcəkdir.

Sorğunun Həqiqiliyini Yoxlamaq

PHP
// 1. Epoint-dən alınan data + private_key ilə signature yarat
$signature = base64_encode(sha1($private_key . $data . $private_key, 1));

// 2. Alınan signature ilə müqayisə et
if ($signature === $received_signature) {
    // Orijinal Epoint cavabı
}

// 3. Data-nı decode et
$result = json_decode(base64_decode($data));

PHP (Laravel)
$payload = [
    'public_key'  => 'public_key',
    'amount'      => 2.50,
    'order_id'    => 'order_id_from_your_system',
    'description' => 'Test payment',
];

$data      = base64_encode(json_encode($payload));
$signature = base64_encode(sha1($private_key . $data . $private_key, 1));

Cüzdanlar (Wallets)

Cüzdan Siyahısı

POST https://epoint.az/api/1/wallet/status
JavaScript
const fetchWallets = async () => {
    const response = await fetch("https://epoint.az/api/1/wallet/status", {
        method: "POST",
        headers: { "Content-Type": "application/json" },
        body: JSON.stringify({ public_key: "your_public_key" })
    });
    const wallets = await response.json();
    renderWalletButtons(wallets);
};
